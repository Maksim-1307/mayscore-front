A. Построение заданной траектории

Ограничение времени1 секунда
Ограничение памяти64Mb
Вводстандартный ввод или input.txt
Выводстандартный вывод или output.txt

Современные системы автоматического управления беспилотных воздушных судов (БВС) самолетного типа преимущественно имеют следующие контуры управления: управление низкого уровня - выход на необходимые значения угловой ориентации (тангаж, крен, курс) и горизонтальной скорости полета БВС, управление высокого уровня – формирование опорной траектории по полетному заданию выданному оператором БВС.

В рамках данной задачи, вам предстоит по входному полетному заданию (ПЗ) – которым является набор промежуточных пунктов маршрута (ППМ) в стартовой системе координат, написать программу на языке программирования Python которая рассчитывает длину опорной траектории для прохождения ППМ, при условии минимизации длины маршрута БВС и выводит её длину на экран.

Считается, что БВС передвигается на постоянной высоте H = const с постоянной крейсерской скоростью V g V_g Vg​ = 120 км/ч , ограничение на максимальный угол крена − 6 0 ∘ ≤ γ ≤ 6 0 ∘ -60^{\circ} \leq \gamma \leq 60^{\circ} −60∘≤γ≤60∘, ускорение свободного падения g = 9.8 м / с 2 м/с^{2} м/с2, значение минимально допустимого радиуса разворота можно определить по формуле: R = V g 2 / ( g ⋅ t a n ( γ ) ) ; R = V_g^2 / (g \cdot tan(\gamma)); R=Vg2​/(g⋅tan(γ)); Начальное положение БВС - A ( x 0 , z 0 ) A(x_0,z_0) A(x0​,z0​) = (0,0) м.

Входные данные на задачу ограничены 3-мя точками, СТ – точка старта, ППМ – промежуточный пункт маршрута с дополнительным критерием прохождения, и КПМ – конечный пункт маршрута.

Критерием прохождения ППМ (2 точка) является попадание полного диаметра действия полезной нагрузки БВС d o b s = 2 ⋅ r o b s d_obs = 2 \cdot r_obs do​bs=2⋅ro​bs в радиус ППМ r n r_n rn​, рекомендуется использовать методы пролета точки с радиусом упреждения (Fillet Path) [1].

Критерием прохождения КПМ (3 точка) является момент пересечения координат КПМ с опорной траекторией БВС.

Формат ввода

На вход программы поступает файл в формате CSV

Данный файл можно легко считать, используя библиотеку pandas, при помощи функции read_csv, на выходе данной функции вас будет сформирован DataFrame следующего вида:

Нулевым индексом является исходное положение БВС (СТ) в столбцах “Z” и “X” приведены соответствующие координаты (всегда равны нулю), в столбце с наименованием “r” приведен радиус действия полезной нагрузки d o b s = 2 ⋅ r o b s d_obs = 2 \cdot r_obs do​bs=2⋅ro​bs, значение отличается в каждом новом тесте.

Первым индексом является ППМ с условием прохождения, в столбцах “Z” и “X” приведены соответствующие координаты значения которых отличаются в каждом новом тесте, в столбце с наименованием “r” приведен радиус зоны интереса ППМ r n r_n rn​, значение которого отличается в каждом новом тесте.

Вторым индексом является КПМ, в столбцах “Z” и “X” приведены соответствующие координаты значения которых отличаются в каждом новом тесте, значение столбца с наименованием “r” на данном индексе всегда равно нулю, ввиду условия задачи.

Формат вывода

На стандартный вывод программы требуется вывести одно число: суммарная длина полученной опорной траектории в м.

Пример:

4193.68

Значение длины полученной опорной траектории всегда выводится с точностью до 2-го знака после запятой (даже если это нули).
